# 🏎️ F1 Era Dominance Analysis

## 📌 Overview
F1 Era Dominance Analysis is a data science project that explores Formula 1 performance trends across drivers, teams, nationalities, and eras using Python. 
The project focuses on extracting insights from historical F1 data through exploratory data analysis, visualizations, and summary metrics.
It also includes an interactive Streamlit dashboard for exploring performance trends dynamically.

---

## 🎯 Key Focus Areas
- Driver performance analysis
- Team dominance across eras
- Yearly performance trends
- Era-based comparison
- Driver consistency heatmaps

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

---

## 📊 Outputs
- Top drivers and teams analysis
- Performance trend charts
- Heatmaps of driver consistency
- Summary metrics (dominant driver, team, era)
