# 📁 Project Structure

This project analyzes Formula 1 performance data across different eras.

---

## 🏁 Structure

f1-era-dominance-analysis/
├── src/        → Python scripts + Streamlit app
├── data/       → Dataset used for analysis
├── outputs/    → Generated plots and results
├── docs/       → Documentation files

---

## 📊 What This Project Does

- Cleans and processes F1 dataset
- Calculates performance metrics
- Identifies top drivers and teams
- Analyzes trends over time
- Groups data by era
- Visualizes insights using charts and heatmaps
- Provides an interactive dashboard
